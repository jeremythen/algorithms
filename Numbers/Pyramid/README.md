# Pyramid

Write a function that accepts a positive number N.

The function should print a pyramid shape with N levels using the # character.
Make sure the pyramid has spaces on both the left and right hand sides.

Given:

```js
pyramid(2);

' # '
'###'
```
```js
pyramid(3);

'  #  '
' ### '
'#####'

```

```js
pyramid(4);

'   #   '
'  ###  '
' ##### '
'#######'

```
